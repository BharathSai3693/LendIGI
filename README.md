# LendIGI

The Loan Management System is a web-based application designed to streamline the process of loan management for both admins and users. Built using Node.js, Express.js, and EJS (Embedded JavaScript), this system allows users to apply for loans, track their applications, and make repayments, while administrators can oversee loan approvals, manage users, and monitor repayment statuses.

Key Features
For Users:
✅ User Registration & Login – Secure authentication for borrowers
✅ Apply for Loans – Submit loan applications with necessary details
✅ View Loan Status – Check approval and repayment progress
✅ Make Repayments – Track and pay outstanding loans
✅ Dashboard – User-friendly interface displaying loan history and status
For Admins:
✅ Admin Authentication – Secure login for authorized personnel
✅ Approve/Reject Loans – Review applications and decide approvals
✅ User & Loan Management – View, edit, or delete user loan details
✅ Track Repayments – Monitor due and paid amounts
✅ Admin Dashboard – Comprehensive summary of all loans and user activity
